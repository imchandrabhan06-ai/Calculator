# 🧮 Python Calculator

A simple command-line calculator built using Python.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero
- Menu-driven interface

## Technologies Used

- Python 3

## Project Structure

```
Calculator/
├── calculator.py
├── main.py
└── README.md
```

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Run:

```bash
python main.py
```
